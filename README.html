NAME

Acme::Prereq::BigDistro - Module for testing CPAN module prerequisites

VERSION

version 0.02

SYNOPSIS

   use Acme::Prereq::BigDistro;

DESCRIPTION

Acme::Prereq::BigDistro does nothing. It's used to test prereqs and emulate a big distribution of modules.

SEE ALSO

Other Acme::Prereq modules: Acme::Prereq::A, Acme::Prereq::B, Acme::Prereq::None

Dist::Zilla::Plugin::PrereqsClean, which is what actually uses this thing.
